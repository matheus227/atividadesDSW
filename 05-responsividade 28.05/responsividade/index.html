Responsividade com HTML e CSS

Aula sobre Media Queries, containers e adaptação de layouts.

O que é Responsividade?

Responsividade é a capacidade de um site se adaptar a diferentes tamanhos de tela.

Um sistema responsivo funciona corretamente em:

  • Celulares
  • Tablets
  • Notebooks
  • Monitores grandes

O que é um Container?

Container é uma estrutura utilizada para organizar elementos dentro da página.

Ele ajuda a:

  • Organizar layouts
  • Controlar largura
  • Centralizar conteúdos
  • Melhorar a estrutura visual

Conceito de Mobile First

Mobile First é uma estratégia onde o desenvolvimento começa pelas telas menores.

Depois o layout é adaptado para tablets e desktops.

Media Queries

Media Queries são regras do CSS utilizadas para alterar estilos dependendo do tamanho da tela.

Exemplo:

@media (max-width: 600px) {

    body {
        background-color: lightblue;
    }

}

Padrões de Quebra de Tela

Alguns tamanhos são muito utilizados para adaptação responsiva: 

320px até 480px  -> Celulares pequenos

481px até 768px  -> Tablets e celulares maiores

769px até 1024px -> Tablets na horizontal e notebooks

1025px ou mais   -> Desktops
