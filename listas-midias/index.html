Lista Ordenada

Passo a passo para criar um site:

  1. Crie uma pasta para o projeto. Ex: meuSite
  2. Crie o arquivo principal do site. (index.html)
  3. Escreva a estrutura básica no arquivo index.html, utilizando ! + enter
  4. crie um novo arquivo style.css
  5. utilize a tag < link rel="stylesheet" href="style.css">
  6. modifique o estilo como bem preferir
  7. para adicionar funcionalidades, crie o arquivo: script.js
  8. vincule o arquivo script.js utilizando a tag script
  9. no arquivo script.js utilize a tag alert("pagina carregada")
  10. salve tudo com (ctrl + s)
